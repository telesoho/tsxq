# 引擎选择与安装指南

您看到的这些文件是针对不同 CPU 型号优化的版本。为了让软件正常运行，请按照以下步骤操作：

## 1. 选择哪个文件？

根据您的电脑配置，建议按以下优先级选择（**推荐直接使用第一项**）：

1.  **首选 (大多数电脑)**: `pikafish-avx2.exe`
    *   适用于绝大多数现代电脑（2013年以后的 Intel 或 AMD Ryzen 处理器）。
    *   性能优秀，兼容性好。

2.  **备选 (老旧电脑)**: `pikafish-sse41-popcnt.exe`
    *   如果您的电脑非常老旧，或者运行上面的版本报错，请使用这个。
    *   速度稍慢，但兼容性最强。

3.  **极客选项**: `pikafish-avx512.exe` / `pikafish-vnni512.exe`
    *   仅适用于支持 AVX-512 指令集的最新高端 CPU。普通电脑运行会直接闪退或报错。

## 2. 安装步骤

1.  **复制文件**: 将您选中的文件（例如 `pikafish-avx2.exe`）复制。
2.  **粘贴并重命名**:
    *   找到本项目目录下的 `resources\bin` 文件夹（我已经为您自动创建好了）。
    *   路径: `c:\Users\telesoho\prjs\my\tsxq\resources\bin\`
    *   将文件粘贴进去，并将文件名修改为 **`pikafish.exe`** (注意必须完全一致，不要保留后缀如 `-avx2`)。

## 3. 验证

启动软件 (`npm run dev`)，如果右侧面板显示 "Engine: Ready"，说明引擎加载成功。
